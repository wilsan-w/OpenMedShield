# OpenMedShield

Open-source proof-of-concept for IoMT, OT, and legacy device cybersecurity monitoring in hospital environments.

## Features

- Asset discovery with NetAlertX
- Network monitoring with Suricata
- SIEM/dashboard layer with Wazuh
- Future PCAP analysis with Malcolm/Zeek

## Project Goal

OpenMedShield is inspired by healthcare cybersecurity platforms such as Asimily.  
It aims to demonstrate visibility, passive monitoring, anomaly detection, and event centralization using open-source tools.

## Stack

- Docker
- NetAlertX
- Suricata
- Wazuh
- Malcolm/Zeek
